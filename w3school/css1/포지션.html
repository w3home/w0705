포지션

position 속성의 값을 relative, absolute, fixed로 지정하면 요소의 위치를 나타내는 속성(left, right, top, bottom)을 지정할 수 있다. position의 속성 값은 static, relative, absolute, fixed로서 기본 값은 static이다.

position:static;

박스의 크기와 위치를 살펴보아야 합니다.

박스의 포지션 속성 값을 바꿔가며 테스트해 보아야 합니다.

포지션을 이해하기 위해서는 시험해보고 관찰해 보는 것이 절대적으로 필요합니다.

position:relative;

기준: 위치 속성을 지정하지 않을 때의 위치(요소 자신의 위치)를 기준으로 left, top을 지정한다.

박스의 크기와 위치를 살펴보아야 합니다.

박스의 포지션 속성 값을 바꿔가며 테스트해 보아야 합니다.

포지션을 이해하기 위해서는 시험해보고 관찰해 보는 것이 절대적으로 필요합니다.

position:absolute;

기준: position을 지정한(relative, absolute, fixed) 조상(부모)을 기준으로 left, top, right, bottom을 지정한다. 만약, 조상(부모) 요소에 position을 지정하지 않으면 뷰포트를 기준으로 left, top, right, bottom이 지정된다.

박스의 크기와 위치를 살펴보아야 합니다.

박스의 포지션 속성 값을 바꿔가며 테스트해 보아야 합니다.

포지션을 이해하기 위해서는 시험해보고 관찰해 보는 것이 절대적으로 필요합니다.

z-index

요소가 position에 의해 겹쳐있을 경우 position 속성 값이 absolute, relative, fixed로 지정된 요소와 지정되지 않은 요소 사이의 깊이(depth)는 지정된 요소가 위로 올라간다. 겹쳐 있는 요소 모두 position이 지정되어 있을 경우 코드 상에 아래쪽에 기록되어 있는 요소가 위로 지정된다. 이때 z-index 속성을 이용하여 요소의 깊이를 조정할 수 있다.

박스의 크기와 위치를 살펴보아야 합니다.

박스의 포지션 속성 값을 바꿔가며 테스트해 보아야 합니다.

포지션을 이해하기 위해서는 시험해보고 관찰해 보는 것이 절대적으로 필요합니다.

left, top, right, bottom

position이 relative, absolute, fixed로 지정되어 있을 때 left, top, right, bottom 속성을 사용할 수 있다. 속성 값은 양수/음수 모두 지정 가능하다. 예를 들어 left의 경우 양수이면 왼쪽에서 오른쪽으로 위치를 지정하고, 음수이면 오른쪽에서 왼쪽으로 위치를 지정한다. top의 경우 양수이면 위에서 아래로, 음수이면 아래에서 위로 위치를 지정한다.

left:30px; top:20px;

left:-30px; top:-20px;

right:30px; bottom:20px;

right:-30px; bottom:-20px;

position:fixed;

position 속성 값 중 relative는 static의 특징을 가지면서 left, top, right, bottom, z-index 속성을 지정할 수 있다. absolute와 fixed가 유사하다.

position:static;

position:relative;

position:absolute;

position:fixed;
