목록

<ul>태그(display:block; list-style-type:disc; margin:1em 0; padding-left:40px;)는 순서가 지정되지 않은 목록을 정의합니다. 목록 항목은 기본적으로 글머리 기호(작은 검은색 원)로 표시됩니다.

<ol>태그(display:block; list-style-type:decimal; margin:1em 0; padding-left:40px;)는 순서가 지정된 목록을 정의합니다. 목록 항목은 기본적으로 숫자로 표시됩니다. 기본적으로 순서가 지정된 목록은 1부터 계산을 시작합니다. 지정된 숫자부터 계산을 시작하려면 start속성을 사용할 수 있습니다. value속성은 목록 항목의 값을 설정합니다. 다음 목록 항목은 해당 숫자에서 증가합니다. 값은 숫자여야 하며 순서가 지정된 목록(<ol>)에서만 사용할 수 있습니다.

<li>태그(display:list-item;)는 목록 항목을 정의합니다.

<ul>과 <ol>태그에는 li태그만 사용할 수 있다. 주의!~

목록 항목(<li>)에는 새 목록과 이미지 및 링크와 같은 다른 HTML 요소가 포함될 수 있습니다.

목록은 중첩될 수 있습니다(목록 내부 목록).

<dl>태그(display:block; margin:1em 0;)는 설명 목록을 정의합니다. 각 목록의 용어(이름)은 <dt>태그(display:block;)로 정의하고, <dd>태그(display:block; margin-left:40px;)로 각각의 용어를 설명합니다. <dd>태그는 문단, 줄 바꿈, 이미지, 링크, 목록을 넣을 수 있습니다.
