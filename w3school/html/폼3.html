폼

목록상자

콤보상자(=입력상자+목록상자)

datalist 요소는 리스트 항목들을 만든다. input 태그의 list 속성은 텍스트 필드에 내용 입력시 미리 만들어 놓은 리스트 항목이 마치 자동완성 기능처럼 나타나도록 하는 속성이다.

폼의 속성

required

required 속성은 입력 필드에 값이 채워졌는지에 대한 부분과 유효성을 판단해주는 속성이다. input 요소에 값을 입력하지 않거나 유효하지 않은 상태로 '전송'버튼을 누르면 오류 메시지 나타난다.

*표시는 필수 입력 항목입니다.

placeholder

placeholder 속성은 각 필드에 대한 입력형식 힌트를 제공한다.

  • 특수문자(-) 없이 숫자로만 입력해주세요.

autocomplete

autocomplete 속성은 자동완성 기능에 대한 사용 여부를 지정한다. 속성 값은 사용하려면 on으로, 사용하지 않으려면 off로 지정한다.

autofocus

autofocus 속성은 페이지가 로드되었을 때 입력 필드에 포커스(커서)가 자동으로 이동하게 해주는 속성이다. autofocus 속성은 하나의 input 요소에만 지정해야한다.

disabled/readonly

disabled 속성은 폼 요소의 내용을 선택할 수 없도록 비활성화 상태로 만드는 속성이다. readonly 속성은 폼 요소의 내용을 선택할 수 없는 상태 즉, 읽기 전송 속성이다.

pattern

pattern 속성은 필드의 값을 체크해 주는 폼 유효성 검사 속성이다. 속성 값은 '정규 표현식'으로 지정하고, 입력 값과 정규식이 일치해야만 유효성에 통과할 수 있다. title 속성으로 입력 형식에 대한 툴팁을 제공해 주는 것이 좋다.

multiple

multiple 속성은 여러 개의 이메일을 ','로 입력하거나 다중의 파일을 선택할 수 있도록 해주는 속성이다.
