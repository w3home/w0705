폼

form 태그의 action 속성은 폼에서 작성한 값을 처리하는 서버 프로그램 주소를 지정하는 속성이다. method 속성은 서버에 값을 전달하는 방식으로 post와 get이 있다. get 방식은 웹브라우저의 주소표시줄에 전달 값이 노출되며, 전송할 수 있는 데이터 양이 적다. post 방식은 전송되는 데이터 값들이 노출되지 않으며 많은 양의 데이터를 전송할 수 있다.

fieldset 요소는 블록 레벨 요소이며, 폼 요소를 그룹화하는데 사용한다. legend 요소는 fieldset 안에 그룹의 제목을 지정하는데 사용한다.

입력 양식(input)

input은 인라인 요소이며 type 속성으로 다양한 폼 컨트롤을 생성할 수 있다.
